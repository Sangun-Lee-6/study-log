# 비전공자도 이해할 수 있는 AWS 입문/실전

AWS EC2로 백엔드 API 서버를 배포하고, 도메인, HTTPS, 데이터베이스, 파일 저장소, 웹 페이지 배포까지 실제 서비스 배포 흐름을 익히는 강의입니다.[링크](https://www.inflearn.com/course/%EB%B9%84%EC%A0%84%EA%B3%B5%EC%9E%90-%EC%9D%B4%ED%95%B4%ED%95%A0%EC%88%98%EC%9E%88%EB%8A%94-aws-%EC%9E%85%EB%AC%B8%EC%8B%A4%EC%A0%84/dashboard?cid=332108)

AWS의 주요 서비스인 EC2, Route53, ELB, RDS, S3, CloudFront를 실습 중심으로 다루며, 각 서비스를 어떤 상황에서 사용하고 서로 어떻게 연결하는지 학습합니다.
