# 시간복잡도

## 📌 입력 크기별 가능한 시간복잡도

| n 최대값          | 보통 가능한 복잡도                   | 예시                 |
| ----------------- | ------------------------------------ | -------------------- |
| `n <= 10`         | `O(n!)`, `O(2^n)`                    | 순열, 완전탐색       |
| `n <= 20`         | `O(2^n)`                             | 부분집합, 비트마스크 |
| `n <= 10^2`       | `O(n^3)`                             | Floyd-Warshall       |
| `n <= 10^3`       | `O(n^2)` 또는 가벼운 `O(n^3)` 일부   | DP, 2중 루프         |
| `n <= 10^4`       | `O(n log n)`, 가벼운 `O(n^2)`은 위험 | 정렬, 힙             |
| `n <= 10^5`       | `O(n log n)`, `O(n)`                 | 투포인터, 해시, 정렬 |
| `n <= 10^6`       | `O(n)`                               | 단순 순회            |
| `n >= 10,000,000` | `O(n)`도 언어/상수에 따라 위험       | 입출력 최적화 필요   |

## 📌 알고리즘 선택 감각

| 조건에서 보이는 신호       | 의심할 알고리즘                       |
| -------------------------- | ------------------------------------- |
| `n <= 20`                  | 백트래킹, 비트마스크                  |
| `n <= 100`                 | 3중 DP, Floyd-Warshall                |
| `n <= 3000`                | `O(n^2)` DP, 투포인터                 |
| `n <= 100000`              | 정렬, 해시, 투포인터, 슬라이딩 윈도우 |
| `n <= 1000000`             | 단순 순회, prefix sum                 |
| `n`이 크고 그래프          | BFS/DFS `O(V+E)`                      |
| 정렬된 배열 또는 정렬 가능 | Binary Search, Two Pointer            |
| 중복 제거/존재 확인        | HashSet, HashMap                      |
